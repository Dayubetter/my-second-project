### lambda 表达式
- java8开始：替换某些匿名内部类对象，从而让程序更简介，可读性更好
```java
(被重写方法的形参列表)->{
    被重写方法的方法体代码
        }
```
- java8开始：lambda表达式的参数列表可以省略，如果只有一个参数，可以省略参数列表
- lambda只能替代函数式接口的匿名内部类  函数式接口：有且仅有一个抽象方法的接口

### 静态方法引用
- 静态方法引用：类名::静态方法名
- 如果某个lambda表达式里只是调用一个静态方法，并且“->” 前后参数的形式一致，就可以使用静态方法引用

### 实例方法引用
- 实例方法引用：对象名::实例方法名
- 如果某个lambda表达式里只是调用一个实例方法，并且“->” 前后参数的形式一致，就可以使用实例方法引用

### 特定类型的方法引用
- 特定类方法引用：类名::静态方法名
- 如果某个Lambda表达式里只是调用一个特定类型的实例方法，并且前面参数列表中的第一个参数作为方法的主调，后面的所有参数都是作为该实例方法的入参的，则此时就可以使用特定类型的方法引用

```java
    import java.util.Arrays;

Arrays.sort(names,(s1, s2) -> s1.comtoIgnoreCase(s2));
Arrays.sort(names,String::compareToIgnoreCase);
```

### 构造器引用
- 构造器引用：类名::new
- 如果某个Lambda表达式里只是调用一个构造器，并且“->” 前后参数的形式一致，就可以使用构造器引用
- 构造器引用的语法格式：类名::new
