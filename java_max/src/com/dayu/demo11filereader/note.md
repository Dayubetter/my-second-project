### FileReader 文件字符输入流
- 以内存为基准，可以把文件中的数据以字符的形式读入到内存中去

|构造器|说明|
|:----|:----|
|`public FileReader(File file)`|创建一个字符输入流，该流从指定的文件中读取字符|
|`public FileReader(String fileName)`|创建一个字符输入流，该流从指定的文件中读取字符|

|方法|说明|
|:----|:----|
|`public int read()`|从输入流中读取一个字符，返回该字符，如果已到达文件末尾，则返回-1|
|`public int read(char[] cbuf)`|从输入流中读取一个字符数组，返回实际读取的字符数，如果已到达文件末尾，则返回-1|
